# Sales Data Analysis Project

## Overview
This project demonstrates a beginner's data analysis using Microsoft Excel. The dataset represents sales transactions across different product categories.

## Objectives
-Calculate total revenue
-Analyse revenue by category
-Visualise category performance using a 2D bar chart

## Tools Utilised
-Microsoft Excel
-SUM function
-SUMIF function
-Basic data visualisation

## Key Insights
-Electronics generated the highest revenue.
-Furniture performed moderately well.
-Stationery had a lower revenue but with higher quantity sold.

## Purpose
This project showcases analytical skills including data calculations, grouping, and visualisation.
